使用 hexo 和 Github 搭建属于你自己网站（自建网站）

1. 系统信息

Windows 版本：Windows 7 企业版，系统类型：64 位操作系统。

2. 必备软件安装和检查：

2.1 安装 Node.js 并配置好 Node.js 环境

  • Node.js 安装和配置此处不做赘述，检查安装，打开 cmd.exe，键入 node -v 和 npm -v，显示如下表示安装成功：
geekpanshi@geek2945 MINGW64 /e/selfDatas $ node -v v8.11.3 geekpanshi@geek2945 MINGW64 /e/selfDatas $ npm -v 5.6.0

2.2 安装 Git 并配置 Git 环境

  • 安装好了 Git，桌面会出现 ，鼠标右键 菜单会出现 Git GUI Here 和 Gui Bash Here 选项。

  • 此时，Git 相关的操作在 Git Bash Here 中进行，假如要在 Windows Cmd.exe 中进行，需要在安装时选择作如下选择

  • 检查安装，打开 cmd.exe，键入 git -v ，显示如下表示安装成功：

5.2 启动服务器

  • $ hexo server 或 简写的 $ hexo s 开启服务器，默认端口号 4000，可用参数 -p number 指定端口号。
geekpanshi@geek2945 MINGW64 /e/selfDatas/demo/geekpanshi $ hexo server -p 5000 INFO Start processing INFO Hexo is running at http://localhost:5000 . Press Ctrl+C to stop.

5.3 测试访问本地

  • 打开浏览器，输入 http://localhost:5000/ 访问本地刚开启的服务器，能够打开，说明安装正常。

6. 设置 Git 和生成本地秘钥对

6.1 设置 Git 的用户和邮箱信息

  • 配置属于自己的用户名和邮箱，下面所示代码中 geekpasnhi 和 geekpasnhi@example.com 替换成自己的。
geekpanshi@geek2945 MINGW64 /e/selfDatas/demo/geekpanshi $ git config --global user.name "geekpasnhi" geekpanshi@geek2945 MINGW64 /e/selfDatas/demo/geekpanshi $ git config --global user.email "geekpasnhi@example.com" geekpanshi@geek2945 MINGW64 /e/selfDatas/demo/geekpanshi $

6.2 生成秘钥对

  • 进入 .ssh 目录，cd ~/.ssh， 假如没有请自行创建目录并进入目录 mkdir -p ~/.ssh && cd ~/.ssh。
  • 查看是否有秘钥 id_rsa 和 id_rsa.pub， ls -al（假设没有）。
  • 生成秘钥(输入 ssh-keygen -t rsa -C “geekpasnhi@example.com”， 邮箱替换成自己的，连续回车 3 次，生成秘钥，并查看秘钥，秘钥路径 ~/.ssh)

7. 登录 Github，绑定生成的秘钥，并验证绑定

7.1 登录 Github，添加 SSH key

  • 登录 Github，点击头像下的 Settings，选择 SSH and GPG keys，点击 New SSH key，把 id_ras.pub 的内容添加 ssh。如下图

7.2 验证是否添加绑定成功

geekpanshi@geek2945 MINGW64 /e/selfDatas/demo/geekpanshi $ ssh -T git@github.com Hi GeekPanshi! You've successfully authenticated, but GitHub does not provide shell access. geekpanshi@geek2945 MINGW64 /e/selfDatas/demo/geekpanshi $

7.3 绑定失败处理

  • 清除所有的绑定关系
geekpanshi@geek2945 MINGW64 /e/selfDatas/demo/geekpanshi $ ssh-add -D All identities removed. geekpanshi@geek2945 MINGW64 /e/selfDatas/demo/geekpanshi $
  • 删除 Github，上面绑定的 Public-key 。
  • 删除目录 ~/.ssh 下面的 id_rsa 和 id_rsa.pub，rm -rf ~/.ssh/id_rsa ~/.ssh/id_ras.pub。
  • 重新执行 6.2、6.3、7.1、7.2 步骤。
