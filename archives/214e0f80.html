Linux Kernel 全局一览

先来看一幅 Linux kernel map

下图是由 makelinux 网站 提供的一幅非常经典的 Linux 内核图，涵盖了内核最为核心的方法. Linux 除了驱动开发外，还有很多通用子系统，比如 CPU、memory、file system 等核心模块，即便不做底层驱动开发， 掌握这些模块对于加深理解整个系统运转机制还是很有帮助。

目录解释部分子目录
